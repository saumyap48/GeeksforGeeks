The task is to find the minimum characters to be added at the front to make string palindrome.</span></p>
<blockquote>
<p><span style="font-size: 18px;">A palindrome is a word that reads the same backwards as forward. Example: "madam".</span></p>
</blockquote>
<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>s = "abc"
<strong>Output: </strong>2
<strong>Explanation: </strong>Add 'b' and 'c' at front of above string to make it palindrome : "cbabc"</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>s = "aacecaaaa"
<strong>Output: </strong>2
<strong>Explanation:</strong> Add 2 a's at front of above string to make it palindrome : "aaaacecaaaa"</span></pre>
